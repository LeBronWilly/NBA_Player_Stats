# NBA_Player_Stats
 NBA Player Traditional Statistics (from 1996-97 to 2022-23)

2. <a href="https://www.flaticon.com/free-icons/nba" title="Basketball Icon">Basketball Icons created by amoghdesign - Flaticon</a>
